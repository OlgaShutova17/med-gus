## Итог MVP

Через 3 месяца продукт умеет:

```
хранить медицинскую историю
структурировать проблему
сохранять анализы
показывать динамику
анализировать результаты
давать рекомендации
```

---

## Реально важная фича MVP

Самая ценная часть продукта:

```
AI + Timeline лечения
```

Пользователь должен видеть:

```
симптом
↓
анализ
↓
врач
↓
лечение
↓
результат
```
