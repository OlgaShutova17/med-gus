---
title: Создание тематики
order: 1
---

Я как пользователь хочу иметь возможность создавать тематику по конкретной проблеме, чтобы отслеживать в ней все события, результаты обследований.

**Цель:**

* Обеспечить структурирование информации по конкретным проблемам

**Предусловие:**

* Пользователь зарегистрирован и авторизован в системе

* Пользователь находится на странице списка тематик

* У пользователя есть доступ к созданию тематик

**Запускающее событие:**

Пользователь нажимает кнопку "Создать тематику" на странице списка тематик.

**Основной сценарий:**

1. Пользователь нажимает кнопку "Создать тематику"

2. Открывается окно для создания тематики с полем "Название тематики" и Описание тематики

3. Пользователь вводит название

4. Пользователь при необходимости заполняет текстовое поле "Описание проблемы"

5. Система проверяет обязательность заполнения поля "Название"

6. Пользователь подтверждает создание

7. Система успешно создает тематику

8. После успешного создания тематика появляется в списке

**Критерии приемки:**

1. Есть кнопка "Создать тематику"
2. Открывается окно для создания тематики с полем "Название тематики"
3. Поле “Название” является обязательным. Если название пустое, ошибка валидации
4. После успешного создания тематика появляется в списке
5. Пользователь переходит внутрь тематики и может добавлять в нее "Симптомы", "Гипотезы" "События" "Результаты обследований"

А**льтернативный сценарий** - Создание тематики из раздела "Анализ?"
