Постусловия: Пользователь видит актуальную динамику анализов.

Как пользователь, я хочу видеть динамику изменения состояния и результатов медицинских анализов, чтобы отслеживать прогресс здоровья и вовремя реагировать на отклонения.

**Критерии приемки**

* Отображение графика изменений ключевых показателей анализов (например, гемоглобин, глюкоза) по датам сдачи;
* Цветовая индикация: зеленый — норма, желтый — отклонение, красный — критическое значение;
* Фильтрация по типу анализа и периоду (неделя, месяц, квартал);
* Возможность просмотра детальной информации по конкретной точке на графике (дата, значение, референсные значения).
* Две синие пунктирные линии, отражающие минимально и максимально допустимые уровни значений для выбранного типа анализа.

---

**Use Case:** Просмотр динамики медицинских анализов

**Название:** Просмотр динамики изменения результатов анализов

**Приоритет:** Высокий

**Акторы:** Пользователь

**Предусловия:** Пользователь авторизован в системе.

**Основной сценарий:**

1. Пользователь входит в раздел "Мои анализы".
2. Система отображает список доступных типов анализов (гемоглобин, глюкоза и т.д.).
3. Пользователь выбирает тип анализа (например, "Гемоглобин").
4. Пользователь выбирает период (неделя, месяц, квартал).
5. Система отображает график изменений показателя по датам сдачи:
6. Пользователь наводит курсор на точку на графике.
7. Система показывает всплывающую подсказку с деталями: дата сдачи, значение, референсные значения.
8. Пользователь закрывает подсказку или выбирает другой анализ/период.

**Альтернативные сценарии:**

A1. Нет данных для выбранного периода:

1. Пользователь выбирает тип анализа и период.
2. Система показывает сообщение: "Анализы за выбранный период отсутствуют"

A2. Один анализ доступен:

1. Система отображает единственную точку с деталями вместо графика.

---

**Бизнес-правила**

1. Определение нормы показателя анализа

   Для каждого типа анализа система использует референсные диапазоны:

   Норма (зеленый): значение в пределах минимума и максимума.

   Отклонение (желтый): значение меньше минимума на максимум 10% либо больше максимума на максимум 10%.

   Критическое (красный):значение меньше минимума на более, чем 10% либо больше максимума на более, чем 10%.
2. Правила фильтрации по периоду

   Неделя: последние 7 дней от текущей даты.

   Месяц: последние 30 дней.

   Квартал: последние 90 дней.
3. Детализация точки графика

   При наведении курсора отображается: дата, значение, норма: мин / макс, статус: норма / отклонение / критическое
4. Правила доступа: пользователь видит только свои анализы
