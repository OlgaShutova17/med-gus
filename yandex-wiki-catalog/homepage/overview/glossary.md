Термины, принятые в проекте.

| Термин | Определение | Пример |
|---|---|---|
| **Тематика** | Раздел, создаваемый пользователем для конкретной проблемы; структурирует всю связанную информацию | «Летняя травма колена», «Мои мигрени», «Подозрение на гастрит» |
| **Событие** | Действие, направленное на решение проблемы. Четыре типа: консультация, анализ, исследование, другое | Приём у терапевта, сдача ОАК, МРТ колена |
| **Консультация специалиста** | Тип события со свободной формой: описание приёма, назначения, направления | «02.03.2026 — пульмонолог, назначены препараты X, направление к кардиологу» |
| **Направления** | Рекомендация врача обратиться к специалисту другого профиля | Терапевт → ЛОР; гастроэнтеролог → эндокринолог |
| **Назначения** | Список манипуляций для стабилизации состояния: лекарства, режим, питание, физактивность | «Препарат X 50 мг курсом 10 дней, ограничить физнагрузку» |
| **Анализы** | Тип события со строгой формой: числовые значения показателей по шаблону | ОАК, биохимия, анализ мочи — с полями «значение / единица / норма» |
| **Исследования** | Тип события по диагностике без числовых значений: текстовое заключение + файл | МРТ, КТ, УЗИ, рентген, ФГДС |
| **Другое** | События, не подходящие под три типа выше — свободная форма | Физиолечение, иглорефлексотерапия |
| **Гипотеза** | Предполагаемая причина проблемы, которую пользователь проверяет через события | «Дефицит железа», «Гипотиреоз», «Хронический стресс» |
| **Резолюция** | Субъективная оценка пользователя по итогу события | «Помогло» / «Не помогло» |
| **Динамика изменений** | Хронологические графики числовых показателей из анализов | График ферритина за последние 6 месяцев |
| **Референсное значение** | Диапазон нормы для физиологического показателя у здоровых людей | Ферритин: 20–150 мкг/л; Гемоглобин: 120–160 г/л |
| **Форма анализа** | Экран ввода результатов анализа: набор полей зависит от типа анализа | Поля: параметр, значение, единица, норма (мин/макс) |
