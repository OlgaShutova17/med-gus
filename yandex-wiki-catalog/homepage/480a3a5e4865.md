| Сущность                 | Атрибуты                                                            |
| ------------------------ | ------------------------------------------------------------------- |
| **ПОЛЬЗОВАТЕЛЬ**         | **ID_ПОЛЬЗОВАТЕЛЯ**, имя, email, дата_регистрации                   |
| **ТЕМАТИКА**             | **ID_ТЕМАТИКИ**, название, описание, дата_создания                  |
| **ГИПОТЕЗА**             | **ID_ГИПОТЕЗЫ**, описание, дата_создания, статус                    |
| **СОБЫТИЕ**              | **ID_СОБЫТИЯ**, дата_события, комментарий                           |
| **КОНСУЛЬТАЦИЯ**         | **ID_КОНСУЛЬТАЦИИ**, врач, клиника, диагноз, заключение             |
| **АНАЛИЗ**               | **ID_АНАЛИЗА**, тип_анализа, дата_проведения                        |
| **ИССЛЕДОВАНИЕ**         | **ID_ИССЛЕДОВАНИЯ**, тип_исследования, дата_проведения, заключение  |
| **ПОКАЗАТЕЛЬ_АНАЛИЗА**   | **ID_ПОКАЗАТЕЛЯ**, название_показателя, значение, единица_измерения |
| **РЕФЕРЕНСНОЕ_ЗНАЧЕНИЕ** | **ID_РЕФЕРЕНСА**, минимальное_значение, максимальное_значение       |
| **РЕЗОЛЮЦИЯ**            | **ID_РЕЗОЛЮЦИИ**, статус_резолюции, комментарий                     |
| **ФАЙЛ**                 | **ID_ФАЙЛА**, имя_файла, тип_файла, путь_к_файлу                    |
| **КУРС_ЛЕЧЕНИЯ**         | **ID_КУРСА**, диагноз, дата_начала, дата_окончания                  |
| **ПРЕПАРАТ**             | **ID_ПРЕПАРАТА**, название_препарата, дозировка                     |
| **НАЗНАЧЕНИЕ**           | **ID_НАЗНАЧЕНИЯ**, описание, дозировка, частота_приёма              |
| **НАПРАВЛЕНИЕ**          | **ID_НАПРАВЛЕНИЯ**, специалист, цель_направления                    |





| Сущность           | Связь       | Сущность             | Кардинальность |
| ------------------ | ----------- | -------------------- | -------------- |
| ПОЛЬЗОВАТЕЛЬ       | создаёт     | ТЕМАТИКА             | **1:N**        |
| ТЕМАТИКА           | содержит    | ГИПОТЕЗА             | **1:N**        |
| ГИПОТЕЗА           | проверяется | СОБЫТИЕ              | **1:N**        |
| СОБЫТИЕ            | имеет       | РЕЗОЛЮЦИЯ            | **1 : 0..1**   |
| СОБЫТИЕ            | содержит    | ФАЙЛ                 | **1:N**        |
| СОБЫТИЕ            | ISA         | КОНСУЛЬТАЦИЯ         | **1:1**        |
| СОБЫТИЕ            | ISA         | АНАЛИЗ               | **1:1**        |
| СОБЫТИЕ            | ISA         | ИССЛЕДОВАНИЕ         | **1:1**        |
| АНАЛИЗ             | содержит    | ПОКАЗАТЕЛЬ_АНАЛИЗА   | **1:N**        |
| ПОКАЗАТЕЛЬ_АНАЛИЗА | имеет       | РЕФЕРЕНСНОЕ_ЗНАЧЕНИЕ | **N:1**        |
| КОНСУЛЬТАЦИЯ       | содержит    | НАЗНАЧЕНИЕ           | **1:N**        |
| КОНСУЛЬТАЦИЯ       | содержит    | НАПРАВЛЕНИЕ          | **1:N**        |
| ПОЛЬЗОВАТЕЛЬ       | проходит    | КУРС_ЛЕЧЕНИЯ         | **1:N**        |
| КУРС_ЛЕЧЕНИЯ       | включает    | ПРЕПАРАТ             | **1:N**        |
