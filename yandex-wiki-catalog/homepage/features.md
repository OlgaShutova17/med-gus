## Фичи

### 1. **Пользовательский модуль**

* **Регистрация и авторизация**

  * Фронт: формы регистрации/авторизации, восстановление пароля

  * Бэк: API для аутентификации, хеширование паролей

  * БД: таблица Users (id, email, password\_hash, created\_at)

* **Профиль пользователя**

  * Фронт: личный кабинет, настройки профиля

  * Бэк: API для управления данными профиля

  * БД: таблица UserProfile (id, user\_id, name, birthdate, gender)

### 2. **Модуль тематик**

* **Создание и управление тематиками**

  * Фронт: интерфейс создания/редактирования тематик

  * Бэк: API для работы с тематиками

  * БД: таблица Themes (id, user\_id, title, description, created\_at)

* **Привязка событий к тематикам**

  * Фронт: система связей между событиями и тематиками

  * Бэк: API для управления связями

  * БД: таблица ThemeEvents (theme\_id, event\_id)

### 3. **Модуль событий**

* **Консультации**

  * Фронт: форма добавления консультации

  * Бэк: API для работы с консультациями

  * БД: таблица Consultations (id, user\_id, date, specialist, description)

* **Анализы**

  * Фронт: интерфейс добавления/просмотра анализов

  * Бэк: API для работы с анализами

  * БД: таблица Analyses (id, user\_id, type, date, results)

* **Исследования**

  * Фронт: форма добавления исследований

  * Бэк: API для работы с исследованиями

  * БД: таблица Research (id, user\_id, type, date, description)

### 4. **Модуль гипотез**

* **Создание гипотез**

  * Фронт: интерфейс формирования гипотез

  * Бэк: API для работы с гипотезами

  * БД: таблица Hypotheses (id, theme\_id, title, status)

* **Проверка гипотез**

  * Фронт: система отслеживания результатов

  * Бэк: API для управления статусами

  * БД: таблица HypothesisStatus (id, hypothesis\_id, status, date)

### 5. **Модуль напоминаний**

* **Система уведомлений**

  * Фронт: интерфейс настроек уведомлений

  * Бэк: сервис отправки уведомлений

  * БД: таблица Notifications (id, user\_id, type, date)

* **Напоминания о приеме лекарств**

  * Фронт: календарь приема

  * Бэк: планировщик напоминаний

  * БД: таблица MedicationSchedule (id, user\_id, medication\_id, schedule)

### 6. **Модуль аналитики**

* **Динамика показателей**

  * Фронт: графики и визуализация данных

  * Бэк: сервис обработки данных

  * БД: таблица Metrics (id, user\_id, type, value, date)

* **Статистика по резолюциям**

  * Фронт: отчеты и аналитика

  * Бэк: агрегация данных

  * БД: таблица Resolutions (id, event\_id, result, comment)

### 7. **Модуль ИИ-компонента**

* **Обработка анализов**

  * Бэк: интеграция с ИИ-сервисами

  * БД: таблица AnalysisAI (id, analysis\_id, ai\_result)

* **Рекомендации**

  * Бэк: сервис формирования рекомендаций

  * БД: таблица Recommendations (id, user\_id, type, description)

### 8. **Модуль файлов**

* **Загрузка документов**

  * Фронт: интерфейс загрузки

  * Бэк: файловое хранилище

  * БД: таблица Files (id, user\_id, type, path, size)

* **Хранение результатов**

  * Бэк: система управления файлами

  * БД: таблица FileLinks (id, file\_id, event\_id)

### Общие компоненты

* **Система прав доступа**

  * Контроль доступа к данным

  * Журналирование действий

* **Поиск и фильтрация**

  * Полнотекстовый поиск

  * Система фильтров

* **История изменений**

  * Отслеживание версий

  * Восстановление данных
