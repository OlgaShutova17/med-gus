---
title: Просмотр динамики анализов
---

## User Story

Как пользователь, я хочу видеть динамику изменения результатов медицинских анализов, чтобы отслеживать прогресс здоровья и вовремя реагировать на отклонения.

## Приоритет

Высокий

## Предусловия

- Пользователь авторизован в системе
- В профиле есть хотя бы один сохранённый анализ с числовыми показателями

## Бизнес-правила

1. **Цветовая индикация норм:**
   - Норма (зелёный): значение в пределах референсного диапазона
   - Отклонение (жёлтый): значение выходит за диапазон не более чем на 10%
   - Критическое (красный): значение выходит за диапазон более чем на 10%
2. **Периоды фильтрации:**
   - Неделя: последние 7 дней
   - Месяц: последние 30 дней
   - Квартал: последние 90 дней
3. **Детализация точки графика:** дата сдачи, значение, норма (мин/макс), статус (норма / отклонение / критическое)
4. Пользователь видит только свои анализы

## Основной сценарий

1. Пользователь переходит в раздел «Мои анализы»
2. Система отображает список доступных типов анализов
3. Пользователь выбирает тип (например, «Гемоглобин»)
4. Пользователь выбирает период (неделя / месяц / квартал)
5. Система отображает график изменений показателя по датам сдачи с цветовой индикацией и двумя пунктирными линиями референсных значений
6. Пользователь наводит курсор на точку графика
7. Система показывает всплывающую подсказку: дата, значение, референсные значения, статус

## Альтернативные сценарии

### A1. Нет данных за выбранный период

1. Пользователь выбирает тип анализа и период
2. Система показывает сообщение: «Анализы за выбранный период отсутствуют»

### A2. Доступен только один анализ

1. Система отображает единственную точку с деталями вместо линейного графика

## Критерии приёмки

- [ ] График отображает изменения ключевых показателей по датам сдачи
- [ ] Цветовая индикация: зелёный — норма, жёлтый — отклонение, красный — критическое
- [ ] Две пунктирные линии отражают минимально и максимально допустимые значения
- [ ] Доступна фильтрация по типу анализа и периоду
- [ ] При наведении на точку отображается детальная информация (дата, значение, референсы)

## GWT-сценарии

### Просмотр динамики

**Given** авторизованный пользователь с несколькими сохранёнными анализами одного типа
**When** он выбирает тип анализа и период
**Then** система отображает график с цветовой индикацией и референсными линиями

### Отсутствие данных

**Given** авторизованный пользователь
**When** он выбирает тип анализа и период, за который нет записей
**Then** система показывает сообщение об отсутствии анализов за выбранный период
