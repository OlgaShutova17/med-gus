---
title: Добавление консультации специалиста
---

## User Story

Как пациент, я хочу добавлять результаты консультаций специалистов, чтобы понимать причинно-следственные связи заболевания и иметь централизованный доступ к назначениям врачей.

## Приоритет

Высокий

## Предусловия

- Пользователь авторизован в системе
- Пользователь посетил приём врача

## Основной сценарий

1. Пользователь переходит в раздел «Мои консультации»
2. Нажимает «Добавить консультацию»
3. Открывается модальное окно с областью загрузки файла и кнопкой «Добавить вручную»
4. Пользователь выбирает файл (лист приёма) через кнопку «Обзор»
5. Нажимает «Сохранить»
6. Данные сохраняются и отображаются в интерфейсе

## Альтернативные сценарии

### A1. Загрузка через Drag & Drop

1. Пользователь открывает модальное окно добавления консультации
2. Перетаскивает файл в область загрузки
3. Нажимает «Сохранить»

### A2. Ручной ввод данных консультации

1. Пользователь нажимает «Добавить вручную»
2. Открывается полноэкранная форма с полями:
   - Клиника
   - Врач (специализация)
   - Анамнез жизни
   - Анамнез заболевания
   - Течение заболевания
   - Диагноз
   - Лечение
   - Рекомендации
   - Заключение
   - Повторная явка (дата)
3. Все поля необязательны — сохранение допускается при любом наборе заполненных полей
4. Пользователь заполняет нужные разделы и нажимает «Сохранить»

## Критерии приёмки

- [ ] Кнопка «Добавить консультацию» кликабельна
- [ ] Доступна загрузка файла форматов JPG/JPEG/PDF
- [ ] Поддерживается Drag & Drop
- [ ] Доступен ручной ввод с перечисленными полями
- [ ] Все поля при ручном вводе необязательны
- [ ] Данные сохраняются и доступны при повторном входе
