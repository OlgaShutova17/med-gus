---
title: Добавление анализа с прикреплением файла
---

## User Story

Как пользователь, я хочу загружать результаты анализов в виде файла, чтобы хранить их в одном месте вместо мессенджеров, почты и галереи телефона.

## Приоритет

Высокий

## Предусловия

- Пользователь зарегистрирован и авторизован в системе
- У пользователя есть результат анализа в виде файла (PDF или фото)

## Бизнес-правила

1. Доступ только авторизованному пользователю
2. Анализ сохраняется только при наличии: типа анализа, даты проведения и хотя бы одного файла результата
3. Тип анализа выбирается из преднастроенного списка; произвольный ввод через опцию «другое»
4. Дата проведения не может быть позже текущей даты
5. Допустимые форматы файлов: PDF, PNG, JPG, JPEG; максимальный размер — 25 МБ
6. Каждый анализ привязан к профилю пользователя и не виден другим пользователям

## Основной сценарий

1. Пользователь переходит в раздел «Анализы»
2. Выбирает режим добавления с прикреплением файла
3. Система отображает форму: тип анализа, дата проведения, поле загрузки файла
4. Пользователь выбирает тип анализа из списка
5. Указывает дату проведения
6. Прикрепляет файл результата (PDF или изображение)
7. Нажимает «Сохранить»
8. Система валидирует обязательные поля, формат и размер файла
9. Система сохраняет запись и привязывает к профилю пользователя
10. Отображается уведомление об успехе, анализ появляется в списке

## Альтернативные сценарии

### A1. Не заполнены обязательные поля

1. Пользователь нажимает «Сохранить» без типа, даты или файла
2. Система подсвечивает проблемные поля с сообщениями («Выберите тип анализа», «Укажите дату», «Прикрепите файл»)
3. Запись не создаётся, форма остаётся открытой

### A2. Неподдерживаемый формат или превышен размер файла

1. Пользователь выбирает файл недопустимого формата или размером >25 МБ
2. Система показывает сообщение о допустимых форматах и ограничении размера
3. Пользователь выбирает корректный файл

## Критерии приёмки

- [ ] Форма отображает поля: тип анализа, дата, файл
- [ ] Принимаются файлы PDF, PNG, JPG, JPEG размером до 25 МБ
- [ ] Дата не может быть в будущем
- [ ] После сохранения анализ отображается в списке с возможностью просмотра содержимого
- [ ] Данные доступны после повторного входа в приложение

## GWT-сценарии

### Успешное добавление анализа

**Given** авторизованный пользователь находится на экране добавления анализа
**When** он выбирает тип, указывает дату и прикрепляет файл, затем нажимает «Сохранить»
**Then** система создаёт запись и показывает её в общем списке анализов

### Валидация обязательных полей

**Given** авторизованный пользователь находится на экране добавления анализа
**When** он нажимает «Сохранить», не указав тип, дату или не прикрепив файл
**Then** система не создаёт запись и подсвечивает незаполненные обязательные поля

### Доступность после повторного входа

**Given** у пользователя ранее сохранены анализы
**When** он выходит из аккаунта и снова входит
**Then** система отображает ранее сохранённые анализы в общем списке
